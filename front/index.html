Тесты:
POST : /api/send-text
Если ID незнакомый, то на бэке создастся новый пользователь с этим ID

Ответ: 
{
    "message": {
        "content": "",
        "timestamp": ""
    }
}
GET : /api/history/{user_id}
Ответ: 
{
    "user_id": {user_id},
    "history": [
        {
            "role": "",
            "content": "",
            "timestamp": ""
        },
    ]
}
GET : /api/history/{user_id}/last
Ответ: 
{
    "user_id" : {user_id},
    "last_message" : {
        "role": "",
        "content": "",
        "timestamp": ""
    }
}
POST : /api/clear-history/{user_id}
Ответ: 
{
    "message" : "История для user {user_id} была очищена"
}
